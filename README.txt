SIR - TP01
----------


Auteur
------

Anne-Sophie SAINT-OMER


Sujet
-----

Ecrire un programme en JAVA permettant de construire la matrice MUT (usagers, thèmes) dans laquelle chaque élément représente le nombre d’accès de chaque usager à un thème donné. 


Développement
-------------

Un fichier Ex1.java

Ce fichier contient :
- Une classe Ex1
- Trois méthodes exécutées dans le main :
	* creerFichierUtilisateurs
	* creerFichierThemes
	* creerMatrice


Chaque méthode créée respectivement un fichier : 
utilisateurs.txt
themes.txt
MUT.txt


Compilation
-----------

javac Ex1.java
java Ex1
